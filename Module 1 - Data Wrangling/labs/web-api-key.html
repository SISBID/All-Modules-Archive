To look up the BRAF gene information we would.

  1. Look at the docs under GET species lookup http://rest.ensembl.org/documentation/info/symbol_lookup.
  2. Wed look at the Example Request: /lookup/symbol/homo_sapiens/BRCA2?content-type=application/json;expand=1 and modify it with our gene like so: /lookup/symbol/homo_sapiens/BRAF?content-type=application/json;expand=1
  3. Wed crete the url using the base url like this: http://rest.ensembl.org//lookup/symbol/homo_sapiens/BRAF?content-type=application/json;expand=1
  4. Wed request the information with fromJSON
braf_info = fromJSON("http://rest.ensembl.org//lookup/symbol/homo_sapiens/BRAF?content-type=application/json;expand=1")
braf_info$id
